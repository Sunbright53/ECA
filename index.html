Overview

Learning Outcomes

  • Build a strong foundation in coding logic: algorithm, sequence, loops, conditionals, debugging, and problem-solving
  • Create interactive projects in Scratch — from guided tasks to independent design and system thinking
  • Understand how games and digital systems work through input, logic, feedback, scoring, and progression
  • Use AI responsibly as a support tool — not a replacement — for thinking, creativity, planning, and communication
  • Transition from block-based coding to text-based programming in Python with confidence
  • Build Python programs using variables, input, conditions, functions, loops, and system design
  • Develop computational thinking, creativity, communication, and presentation skills through project-based learning

My Blocks Code Example:

go to x: 0 y: 0
set speed to 5

define hit paddle
change score by 1
change speed by 0.3

define stop

Core Rule:

AI is a support tool, not a replacement for thinking.

  • Think before prompting
  • Question AI outputs — is this correct? why?
  • Take ownership of your work
  • Use AI responsibly and safely
Big Picture · Final Outcome
  • Think in systems — not just write code
  • Design and build real digital experiences
  • Use AI responsibly and effectively — as a tool, not a crutch
  • Transition across platforms: Scratch → Python → Web
  • Communicate and present their ideas with confidence
